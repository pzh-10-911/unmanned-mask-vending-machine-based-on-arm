## 逐元件接线表

### 1. RGB-LED — 三色状态/出货指示灯

**说明：** 共阴极 RGB-LED，四个引脚（R、G、B、公共阴极）。每路串 220Ω 限流后接 GPIO。

| 树莓派侧 | 导线色 | → | 元件侧 | 备注 |
|---------|--------|---|--------|------|
| GPIO 17 (pin 11) | 红 | → | 220Ω → RGB-LED R 脚 | 红色通道，货道A |
| GPIO 18 (pin 12) | 绿 | → | 220Ω → RGB-LED G 脚 | 绿色通道，货道B |
| GPIO 19 (pin 35) | 蓝 | → | 220Ω → RGB-LED B 脚 | 蓝色通道，货道C |
| GND (任一) | 黑 | → | RGB-LED 公共阴极 | 共阴极接 GND |

**重要：** 三路各串一个 220Ω，不可只在公共脚串一个。GPIO 输出 HIGH 点亮对应通道。

**颜色-状态映射：**

| 状态 | R | G | B | 颜色 | 说明 |
|------|---|---|---|------|------|
| 待机 (IDLE) | 0 | 1 | 0 | 🟢 绿 | 常亮，系统运行正常 |
| 已选 (SELECTED) | 1 | 1 | 0 | 🟡 黄 | 等待确认 |
| 货道A 出货中 | 1 | 0 | 0 | 🔴 红 | 成人口罩 |
| 货道B 出货中 | 0 | 1 | 0 | 🟢 绿 | 儿童口罩 |
| 货道C 出货中 | 0 | 0 | 1 | 🔵 蓝 | N95 口罩 |
| 缺货/错误 | 1 | 0 | 1 | 🟣 紫 | 闪烁提示 |

---

### 2. 蜂鸣器 — 出货提示音

**说明：** 3 脚无源蜂鸣器模块（VCC / IO / GND），自带驱动电路，无需串电阻。PWM 方波驱动。

| 树莓派侧 | 导线色 | → | 元件侧 | 备注 |
|---------|--------|---|--------|------|
| 3.3V (pin 1 或 pin 17) | 红 | → | 蜂鸣器模块 VCC | 电源供电 |
| GPIO 26 (pin 37) | 灰 | → | 蜂鸣器模块 IO | PWM 信号输入 |
| GND (任一) | 黑 | → | 蜂鸣器模块 GND | 共地 |

**频率参考：** 500Hz(低沉)、1000Hz(默认)、2000Hz(尖锐)。

---

### 3. 红外传感器 — 取货口检测

**说明：** 3 线制红外避障传感器（VCC / GND / OUT）。安装在取货口上方，检测距离调至 5-10cm。

| 树莓派侧 | 导线色 | → | 元件侧 | 备注 |
|---------|--------|---|--------|------|
| 3.3V (pin 1) | 红 | → | 传感器 VCC | **必须用 3.3V，禁止用 5V** |
| GND (任一) | 黑 | → | 传感器 GND | 共地 |
| GPIO 23 (pin 16) | 绿 | → | 传感器 OUT | 遮挡=HIGH，无遮挡=LOW |

**注意：** 传感器通常带电位器可调节检测距离。若实际电平行为相反（遮挡输出 LOW），在代码 `ir_sensor.py` 的 `is_blocked()` 内取反即可。

---

### 4. 投币按键 — 模拟投币

**说明：** 轻触按键，一端接 GPIO，另一端接 GND。启用内部上拉，按下时产生下降沿中断。

| 树莓派侧 | 导线色 | → | 元件侧 | 备注 |
|---------|--------|---|--------|------|
| GPIO 27 (pin 13) | 白 | → | 按键一脚 | 内部上拉已启用 |
| GND (pin 14) | 黑 | → | 按键另一脚 | 同排两脚内部连通，任选一脚 |

**波形说明：**
```
GPIO27 电平: ━━━━━━━━┓        ┏━━━━━━━━
                      ┃ 按下   ┃
                      ┗━━━━━━━━┛
                      ↑ 下降沿触发中断
```

## 电源与共地汇总

| 电源轨 | 连接对象 | 说明 |
|-------|---------|------|
| 3.3V (pin 1) | 蜂鸣器模块 VCC + 红外传感器 VCC | 共用 3.3V，总电流 < 50mA |
| GND (pin 6/9/14/20/25/30/34/39) | RGB-LED 公共阴极 + 蜂鸣器 GND + 红外 GND + 按键一脚 | **全部共地** |

**共地连接示意图：**
```
树莓派 GND ──┬── RGB-LED 公共阴极
             ├── 蜂鸣器模块 GND
             ├── 红外传感器 GND
             └── 投币按键一脚
```

## config.py 引脚定义对照

```python
# ========== 引脚号（BCM） ==========
RGB_R_PIN = 17       # 红色通道 — 货道A 成人口罩
RGB_G_PIN = 18       # 绿色通道 — 货道B 儿童口罩
RGB_B_PIN = 19       # 蓝色通道 — 货道C N95
IR_SENSOR_PIN = 23   # 红外传感器 — 取货口检测
BUZZER_PIN = 26      # 蜂鸣器模块 — PWM 驱动
COIN_BUTTON_PIN = 27 # 投币按键 — 下降沿中断

# GPIO22 空闲备用（原系统状态LED功能由RGB-LED颜色状态替代）
```

## 接线检查清单

上电前逐项确认：

- [ ] RGB-LED 三路各串一个 220Ω 电阻
- [ ] RGB-LED 公共阴极接 GND（非 3.3V）
- [ ] 蜂鸣器 VCC 接 3.3V（非 5V）
- [ ] 红外传感器 VCC 接 3.3V（非 5V）
- [ ] 所有元件 GND 共地
- [ ] VCC/GND 无接反（用万用表通断档测试短路）
- [ ] 杜邦线连接牢固，无虚接
- [ ] 按键内部上拉已启用（代码中 `PUD_UP`），无需外接上拉电阻
